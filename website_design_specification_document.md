# MIRAINA ブランド・デザインシステム定義書

## 1. デザインコンセプト：Digital Editorial（デジタル・エディトリアル）

装飾過多なWebトレンドとは一線を画し、**「文字（言葉）の力」と「余白（間）」**で信頼感を醸成する**雑誌的レイアウト（エディトリアルデザイン）**を採用しています。

* **キーワード:** 知的(Intellectual)、静寂(Serenity)、信頼(Trust)、本質的(Essential)
* **デザインの方向性:**
    * **No Photo, Just Typography:** フリー素材のような写真は一切排除し、文字のジャンプ率と配置の美しさだけで世界観を作る。
    * **Judicious Efficiency（賢明な効率化）:** デザイン自体も無駄を削ぎ落とし、必要な情報だけが美しく浮かび上がる構成。
    * **余白の活用:** 余白をうまく使って洗練されて余裕のあるデザイン配置をする。

## 2. カラーパレット（Color Scheme）

色彩心理に基づき、極めて限定的な色数で「プロフェッショナルな冷徹さ」と「人間味のある温かみ」のバランスを取っています。

| 色名 | カラーコード | 用途・意図 |
| :--- | :--- | :--- |
| **MIRAINA Black** | `#1a1a1a` | **主色（テキスト・背景）。**完全な黒（#000）ではなく、わずかにグレーを含ませることで、画面上のコントラストを和らげ、長時間読んでも疲れない「インクのような黒」を表現。 |
| **Intellect Blue** | `#0052cc` | **アクセントカラー。**「知性」と「テクノロジー」を象徴する鮮やかな青。リンク、ホバー時、重要なポイントにのみ限定的に使用し、視線を誘導する「差し色」として機能。 |
| **Paper Gray** | `#f4f4f4` | **ベースカラー（背景）。**真っ白な画用紙ではなく、質感のある上質紙のような極薄いグレー。セクションごとの区切りや、カードの背景に使用し、奥行きを作る。 |
| **Text Gray** | `#333333` | **本文色。**可読性を最大限に高めたダークグレー。 |

## 3. タイポグラフィ（Typography）

「論理（明朝）」と「機能（ゴシック）」を使い分けることで、情緒と実用性を両立させています。

* **見出し（Headings）: Shippori Mincho（しっぽり明朝）**
    * **役割:** ブランドの品格、伝統、論理的な思考力の表現。
    * **特徴:** 筆脈を感じさせる流麗な明朝体により、AIという無機質な技術に「日本的な情緒」と「人間の意思」を付与します。
* **本文（Body）: Noto Sans JP**
    * **役割:** 機能性、可読性、モダンさ。
    * **特徴:** 長文でも読みやすいユニバーサルなフォント。明朝体の見出しとの対比で、画面にリズムを生みます。
* **欧文・あしらい: Letter Spacing（文字間）**
    * 英語の装飾テキスト（例：`Strategic AI Partner`）は、文字間を極端に広く（`tracking-widest`）取ることで、高級ブランドのような佇まいを演出しています。

## 4. レイアウトと余白（Layout & Whitespace）

* **グリッドシステムの意図的な崩し:**
    * 12カラムグリッドを基本としつつ、左右対称（シンメトリー）ではなく、あえて左右非対称に配置。左に大きな余白を作り、右にコンテンツを寄せるといった「崩し」によって、緊張感とモダンな印象を与えます。
* **ブリージング・スペース（呼吸する余白）:**
    * セクション間のパディング（`py-32`等）を通常のWebサイトの倍以上確保。この「無駄」に見える余白こそが、見る人に「余裕」と「自信」を感じさせます。
* **縦書きの導入:**
    * 装飾として一部に「縦書きテキスト」を配置。視線の流れを一時的に止め、Webサイトでありながら「紙媒体」のような質感を与えます。

## 5. UIコンポーネントとアイコン

* **Thin Line Icons（極細線アイコン）:**
    * `stroke-width: 1.5` の極めて細いSVGアイコンを使用。アイコン自体を主張させず、あくまで「文字情報の補助」として控えめに配置。色は黒か青のみ。
* **Micro-Interaction（微細な動き）:**
    * **リンク:** 下線が左から右へ伸びるアニメーション。
    * **メガメニュー:** ふわりと降りてくる透過と移動の演出。
    * **フェードイン:** ユーザーのスクロールに合わせて、コンテンツが下からゆっくりと浮き上がる（`fade-in-section`）。
    これらは「派手さ」ではなく「丁寧な振る舞い」を感じさせるための演出です。

## 6. indexページにおけるサービス詳細
- サービスカードの背景に適切にデザインされた画像を配置する。ただし、その上のレイヤーに色を重ねて画像が透けて少しだけ見えるくらいに透明度を下げておく
- そのカードでのマウスオーバーとして、背景透明度を上げて背景写真が少し見えやすいようにする
